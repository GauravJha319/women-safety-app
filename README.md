# 🚨 Women Safety Web Application

A Flask-based Women Safety Web Application designed to help users send emergency SOS alerts with live location tracking.

## 🔹 Features

- User Registration & Login
- Secure Session Management
- SOS Button with Live Location
- Emergency Alert Stored in Database
- Alert History Page
- Professional Responsive UI
- GitHub Version Control

## 🔹 Tech Stack

- Python (Flask)
- SQLite Database
- HTML
- CSS
- JavaScript
- Git & GitHub

## 🔹 How It Works

1. User registers with mobile number.
2. Login to dashboard.
3. Press SOS button.
4. Live location is captured.
5. Alert is saved in database.
6. User can view alert history.
